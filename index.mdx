---
title: "MVSoftware"
description: "An integration platform for field service businesses, connecting Vantaca, Jobber, ClickUp, QuickBooks, and Microsoft Teams."
---

MVSoftware is a multi-module operations platform built on Next.js 15 that automates work order synchronization, time tracking, job scrubbing, and receipt analysis for field service businesses.

## Modules

<Columns>
  <Card title="Vantaca-Jobber sync" icon="arrows-rotate" href="/vantaca-jobber">
    Synchronize work orders between Vantaca HOA management and Jobber field service, including attachments and status updates.
  </Card>
  <Card title="TimeTrack" icon="clock" href="/timetrack">
    Pull time entries from ClickUp, classify billability with AI, route approvals, and export to QuickBooks.
  </Card>
  <Card title="Job scrubbing" icon="list-check" href="/scrubbing">
    Review completed Jobber jobs through an 11-stage Kanban workflow from field completion to invoicing.
  </Card>
  <Card title="Receipt analysis" icon="receipt" href="/receipt-analysis">
    Extract line items from BILL platform exports with AI, track vendor pricing, and generate van stocking recommendations.
  </Card>
  <Card title="Teams bot" icon="bell" href="/teams-bot">
    Send real-time notifications to Microsoft Teams for approval queues, sync failures, and scheduled digests.
  </Card>
</Columns>

## Tech stack

| Layer | Technology |
|-------|-----------|
| Framework | Next.js 15 (App Router, Turbopack) |
| Language | TypeScript (strict mode) |
| Database | Supabase (PostgreSQL) |
| UI | Tailwind CSS 4, shadcn/ui |
| Auth | NextAuth.js with Azure AD |
| AI | Anthropic Claude (classification), Voyage AI (embeddings) |
| Monitoring | Sentry with structured logging |
| Testing | Vitest, MSW, jsdom |
| Deployment | Vercel |
