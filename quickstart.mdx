<Info>
  **Prerequisites:**
  - Node.js 22 or higher
  - npm 10 or higher
  - A Supabase project
  - Azure AD tenant with app registration
</Info>

## Install and run

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone <your-repo-url> mvsoftware
    cd mvsoftware
    ```
  </Step>
  <Step title="Install dependencies">
    ```bash
    npm install
    ```
  </Step>
  <Step title="Configure environment variables">
    Copy the example file and fill in your values:

    ```bash
    cp .env.example .env.local
    ```

    At minimum, you need the **Core** variables to start the app. See [environment variables](/environment-variables) for the full reference.

    ```bash
    # Required for all functionality
    NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
    NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
    SUPABASE_SERVICE_KEY=your-service-key
    NEXTAUTH_URL=http://localhost:3000
    NEXTAUTH_SECRET=your-32-char-secret
    CRON_SECRET=your-cron-secret
    ```
  </Step>
  <Step title="Start the dev server">
    ```bash
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000) to see the dashboard.
  </Step>
</Steps>

## Available scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server with Turbopack |
| `npm run build` | Create production build |
| `npm start` | Start production server |
| `npm run validate` | Run type checking and linting |
| `npm test` | Run unit tests with Vitest |
| `npm run test:coverage` | Run tests with coverage report |
| `npm run test:e2e` | Run end-to-end tests |

## Validate before committing

Always run validation before committing changes:

```bash
npm run type-check && npm run lint
```

This catches TypeScript and ESLint errors before they reach CI.
