## Adding new components

Use the shadcn/ui CLI to add components:

```bash
npx shadcn@latest add <component-name>
```

This generates a new file in `components/ui/` following the project's existing patterns (New York style, oklch colors, `data-slot` attributes).

## Form integration

Forms use React Hook Form with Zod schema validation:

```typescript
import { useForm } from "react-hook-form"
import { zodResolver } from "@hookform/resolvers/zod"
import { z } from "zod"

const schema = z.object({
  name: z.string().min(1),
  email: z.string().email(),
})

const form = useForm({
  resolver: zodResolver(schema),
})
```

## Chart components

Data visualizations use Recharts with the theme-aware chart colors (`--chart-1` through `--chart-5`). Charts are used in:
- TimeTrack dashboard summaries
- AI classifier accuracy trends
- Receipt analysis pricing charts
- Approval queue metrics
